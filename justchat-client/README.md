## Just Chat Client

This is the Just Chat Client project, is very simple, my effort was on backend and bot sides.

## Requirements

- Node LTS (v14.x.x)
- Just Chat server running.

## Running

You can run this app as following:

Step 1: Clone the repo

```bash
git clone https://github.com/angelobartolome/justchat
```

Step 2: Build the Docker instance

```bash
cd justchat/justchat-client
npm install
```

Step 3: Run the client in development mode

```bash
npm run start
```
